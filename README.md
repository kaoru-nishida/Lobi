## 必要なソフトなど

https://www.python.org/downloads の2.71.4をインストールしてください</br>
インストールが終わったら、</br>

`pip install requests`

でRequestsモジュールをインストールしてください</br>
インストールが終われば</br>
C:\python27\python.exe <start.pyのパス></br>
で実行できるようになります</br>
