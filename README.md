## 必要なソフトなど

https://www.python.org/downloads/　の2.71.4をインストールしてください
インストールが終わったら、
       pip install requests 
でRequestsモジュールをインストールしてください
インストールが終われば
C:\python27\python.exe <start.pyのパス>
で実行できるようになります
